インストールプロファイルは、Drupalの基本インストールが完了した後に実行される追加の手順を定義します。
また、追加の機能を提供したり、サイトの動作を変更したりすることもあります。

このディレクトリには、ダウンロードしたインストールプロファイルやカスタムのインストールプロファイルを配置します。
なお、インストールプロファイルは通常、Drupalディストリビューションの一部として提供されます。

追加のディストリビューションのダウンロード
----------------------------------------

Drupalコミュニティからの貢献ディストリビューションは、以下のURLからダウンロードできます：
https://www.drupal.org/project/project_distribution.

マルチサイトの設定
------------------

マルチサイトの構成では、このディレクトリにあるインストールプロファイルは、各サイトの初期サイトインストール時に利用できます。

詳細情報
----------

カスタムプロファイルを使用してDrupalを拡張する方法については、Drupalのルートディレクトリの README.md の「インストールプロファイル」セクションを参照してください。
